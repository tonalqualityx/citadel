### Purpose

Quick task creation from anywhere in the application. Slides in from the right side, allowing users to create a quest while maintaining context of their current view.

### Entry Points

| Entry Point | Pre-filled Fields |
|-------------|-------------------|
| Global "+ Add Quest" button | None |
| Pact detail page | Pact, Site, Patron |
| "What's Next" empty state | Current user as assignee |
| Keyboard shortcut (Q) | None |


- Patron dropdown: All active clients, searchable
- Site dropdown: Filters to selected Patron's sites
- "+ Create new site" opens modal for quick site creation
- PM dropdown: Users with Project Manager function
- "Change" link returns to Screen 2

### Validation
- Required: Pact Name, Patron, Site, Project Manager
- Next button disabled until all required fields filled
