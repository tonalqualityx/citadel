## ND Optimizations Applied

| Principle | Implementation |
|-----------|----------------|
| **Roll-up visibility** | Health, progress, time all summarized at container level |
| **Attention focusing** | "Needs Attention" surfaces blocked/overdue/review items |
| **Context preservation** | Breadcrumbs + sidebar links maintain hierarchy awareness |
| **Reduced hunting** | Everything about a Pact/Patron in one place |
| **Clear progress** | Phase dots, progress bars, completion counts throughout |
| **No anxiety colors** | Amber for warnings, never red |
| **Predictable layout** | Consistent section structure across both views |
